# CSI-Based-Sign-Language-Recognition-Using-CNN-GRU-Architecture-Enhanced-with-Attention
CSI-based sign language recognition using a hybrid CNN–BiGRU–Attention model. Extracts spatial–temporal features from Wi-Fi signals without cameras, enhanced by data augmentation and contrastive pretraining. Achieves up to 99% accuracy on Home/Lab datasets with interpretable Grad-CAM insights.
